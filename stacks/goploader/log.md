# goploader

- **Source**: https://github.com/Depado/goploader
- **Category**: File Transfer - Single-click & Drag-n-drop Upload
- **Description**: Easy file sharing server written in Go. Simple drag-and-drop upload with web setup interface.
- **Image(s)**: `daftaupe/goploader:latest`
- **Compose source**: Based on upstream README docker instructions
- **Notes**: Community Docker image (no official image). Setup web interface available at first run on port 8080. Data stored in named volume.
