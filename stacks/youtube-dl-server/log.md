# YoutubeDL-Server

- **Source**: https://github.com/nbr23/youtube-dl-server
- **Image**: nbr23/youtube-dl-server:latest
- **Port**: 8080
- **Description**: Web and REST interface for downloading YouTube videos using yt-dlp. Built with Starlette, supports parallel downloads and job history.
- **Notes**: Config via config.yml (mounted read-only). Supports custom output templates, subtitle downloads, and multiple format presets. SQLite metadata DB.
