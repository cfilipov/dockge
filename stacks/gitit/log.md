# Gitit Stack

## Source
- GitHub: https://github.com/jgm/gitit
- Docker Hub: hyzual/gitit

## Notes
- Gitit is a wiki backed by a Git, Darcs, or Mercurial filestore
- Written in Haskell, uses Pandoc for markup conversion
- Community Docker image (hyzual/gitit)
- Data stored in /data inside the container
- Default port 5001
