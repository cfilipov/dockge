# OnionShare

- **Source**: https://github.com/onionshare/onionshare
- **Category**: File Transfer - Single-click & Drag-n-drop Upload
- **Description**: Securely and anonymously share files, host websites, and chat using the Tor network.
- **Image(s)**: `pataquets/onionshare:latest` (community Docker image)
- **Compose source**: Created from Docker Hub community image
- **Notes**: OnionShare is primarily a desktop application (Windows/macOS/Linux). No official Docker image exists. This uses a community image (`pataquets/onionshare`) which provides headless file sharing over Tor. Functionality may be limited compared to the desktop GUI.
