# Seerr

- **Source**: https://github.com/seerr-team/seerr
- **Image**: seerr/seerr:latest
- **Port**: 5055
- **Description**: Media request management for Jellyfin, Plex, and Emby. Integrates with Sonarr/Radarr. Successor to Overseerr/Jellyseerr.
- **Notes**: Supports PostgreSQL and SQLite. API docs at /api-docs. Mobile-friendly UI with granular permissions.
