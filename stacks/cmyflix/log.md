# cmyflix

## Overview
cmyflix is a simple, self-hosted media catalog for movies and TV shows.

## Image
- `farfalleflickan/cmyflix:latest` from Docker Hub

## Ports
- 5000 → 5000 (web UI)

## Volumes
- `./config` — application config
- Movies and TV bind mounts (read-only)
