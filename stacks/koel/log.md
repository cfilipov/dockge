# Koel

- **Source**: https://github.com/koel/koel
- **Docker image**: phanan/koel
- **Reference**: https://github.com/koel/docker/blob/master/docker-compose.mysql.yml
- **Description**: Personal music streaming server built with Laravel and Vue.js. Requires MariaDB/MySQL backend.
- **Ports**: 8080 (web UI)
- **Volumes**: music, cover images, search indexes, MariaDB data
