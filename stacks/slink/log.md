# Slink Stack

## Source
https://github.com/andrii-kryvoviaz/slink

## Description
Self-hosted image sharing platform built with Symfony and SvelteKit. Supports multi-file upload, image compression, collections, tags, comments, bookmarking, ShareX integration, and S3 storage.

## Services
- **slink** - Main application (anirdev/slink:latest) on port 8080
- **slink-db** - PostgreSQL 16 database

## Reference
Docker image from Docker Hub (anirdev/slink). Symfony backend with SvelteKit frontend.
