# Medusa

- **Source**: https://github.com/pymedusa/Medusa
- **Image**: `lscr.io/linuxserver/medusa:latest` (LinuxServer)
- **Description**: Automatic Video Library Manager for TV Shows (pymedusa). Watches for new episodes and downloads them automatically. NOT medusajs/nextjs-starter-medusa (e-commerce).
- **Ports**: 8081 -> 8081
- **Volumes**: config, downloads, tv shows
- **Key env vars**: PUID, PGID, TZ
- **Category**: Media Management
