# TubeSync

- **Source**: https://github.com/meeb/tubesync
- **Image**: ghcr.io/meeb/tubesync:latest
- **Port**: 4848
- **Description**: PVR for YouTube — like Sonarr but for YouTube channels/playlists. Built-in yt-dlp + ffmpeg download client with task scheduler and retry logic.
- **Notes**: Supports Jellyfin and Plex media server integration. Downloads to /downloads with video/ and audio/ subdirectories. Uses 30m stop grace period for graceful shutdown.
