# Uploady Stack

## Source
https://github.com/farisc0de/Uploady

## Description
Modern secure file uploader with multi-file upload, drag-and-drop, role-based access, admin panel, ClamAV scanning, auto-delete, abuse reporting, and REST API.

## Services
- **uploady** - Main PHP application (farisc0de/uploady:latest) on port 80
- **uploady-db** - MySQL 8.0 database

## Reference
PHP 8.x + MySQL application with Bootstrap 5.3 UI.
