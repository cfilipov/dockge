# TileServer GL

- **Source**: https://github.com/maptiler/tileserver-gl
- **Description**: Vector and raster map tile server with GL styles, server-side rendering by MapLibre GL Native
- **Compose reference**: Constructed from README Docker examples
- **Services**: tileserver (Node.js tile server)
- **Default port**: 8080
- **Notes**: Place MBTiles files in /data volume; supports WMTS, TMS, and GL style rendering
