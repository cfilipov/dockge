# Gollum Stack

## Source
- GitHub: https://github.com/gollum/gollum
- Docker Hub: gollumwiki/gollum

## Notes
- Gollum is a simple Git-backed wiki with a sweet API and local frontend
- Powers GitHub Wikis
- Written in Ruby, supports multiple markup formats (Markdown, MediaWiki, etc.)
- Wiki data stored as a Git repository in /wiki
- Default port 4567
