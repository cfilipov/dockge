# Gerbera

## Overview
Gerbera is a UPnP/DLNA media server that streams digital media to networked devices.

## Image
- `gerbera/gerbera:latest` (official)

## Network
- Host mode (UPnP requires multicast). Default web UI on port 49152.

## Volumes
- `./config` — Gerbera configuration
- Media bind mount (read-only)
