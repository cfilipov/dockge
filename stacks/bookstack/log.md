# BookStack Stack

## Source
- GitHub: https://github.com/BookStackApp/BookStack
- Docker Hub: linuxserver/bookstack (lscr.io/linuxserver/bookstack)

## Notes
- BookStack is a platform for organizing and storing information
- Uses LinuxServer.io image with LSIO conventions (PUID/PGID/TZ)
- Requires MariaDB/MySQL backend
- Default admin credentials: admin@admin.com / password
- Default port 6875 (LSIO convention)
