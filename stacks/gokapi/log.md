# Gokapi

- **Source**: https://github.com/Forceu/gokapi
- **Category**: File Transfer - Single-click & Drag-n-drop Upload
- **Description**: Modern, self-hosted alternative to Firefox Send with automatic file expiration, end-to-end encryption, and cloud storage support.
- **Image(s)**: `f0rc3/gokapi:latest`
- **Compose source**: Based on upstream README docker run command
- **Notes**: Single container with web setup wizard at `/setup` on first run. Data and config stored in separate named volumes. Default port is 53842.
