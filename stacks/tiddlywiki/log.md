# TiddlyWiki Stack

## Source
- GitHub: TiddlyWiki/TiddlyWiki5
- Docker Hub: elasticdog/tiddlywiki

## Description
TiddlyWiki is a unique non-linear notebook for capturing, organising and sharing complex information. This Node.js-based server edition allows multi-user access via HTTP.

## Volumes
- `tiddlywiki-data` — wiki tiddlers and configuration

## Ports
- 8080 → 8080 (HTTP)
