# Judge0 CE

- **Status**: ok
- **Source**: https://github.com/judge0/judge0
- **Image**: judge0/judge0:latest
- **Notes**: Online code execution system supporting 60+ programming languages. Requires privileged mode for container isolation. Uses PostgreSQL for persistence and Redis for job queuing. Configuration is via judge0.conf file (mounted read-only). API available on port 2358.
