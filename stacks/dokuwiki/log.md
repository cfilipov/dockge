# DokuWiki Stack

## Source
- GitHub: https://github.com/dokuwiki/dokuwiki
- Docker Hub: linuxserver/dokuwiki (lscr.io/linuxserver/dokuwiki)

## Notes
- DokuWiki is a simple, versatile wiki that doesn't require a database
- Uses LinuxServer.io image with LSIO conventions (PUID/PGID/TZ)
- File-based storage (no database needed)
- Complete setup wizard on first access
- Default ports: 8080 (HTTP), 8443 (HTTPS)
