# AmuseWiki Stack

## Source
- GitHub: https://github.com/melmothx/amusewiki
- Docker Hub: rojen/amusewiki

## Notes
- AmuseWiki is a library-oriented wiki engine based on the Text::Amuse markup
- Community Docker image (rojen/amusewiki); no official image available
- Data stored in /var/lib/amusewiki inside the container
- Default port 8080
