# Beets

- **Source**: https://github.com/beetbox/beets
- **Docker image**: lscr.io/linuxserver/beets (LinuxServer.io)
- **Reference**: https://github.com/linuxserver/docker-beets
- **Description**: Music library manager and tagger with web interface. Organizes music collection, fixes tags, and fetches metadata.
- **Ports**: 8337 (web UI)
- **Volumes**: config, music library, downloads/ingest
