# TagSpaces

## Source
- GitHub: https://github.com/tagspaces/tagspaces
- Docker Hub: https://hub.docker.com/r/tagspaces/tagspaces-lite-web

## Description
An offline, open source, document manager with tagging support. Helps organize files on any platform using tags rather than folder hierarchies.

## Stack
- tagspaces/tagspaces-lite-web:latest — web lite version (port 80)

## Notes
- TagSpaces Lite Web is the self-hosted web version
- Supports file tagging, previewing, and organizing
- Data stored in /data volume
- Also available as desktop and mobile apps
