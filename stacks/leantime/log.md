# Leantime

- **Source**: https://github.com/Leantime/leantime
- **Image**: leantime/leantime:latest (official)
- **Status**: created
- **Notes**: Open source project management system for small teams and startups. Compose based on official docker-leantime repository (https://github.com/Leantime/docker-leantime). Uses MySQL 8.4 backend with health checks. Includes security hardening (no-new-privileges, minimal capabilities).
