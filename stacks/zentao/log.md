# ZenTao

- **Status**: created
- **Source**: https://www.zentao.pm/
- **Image**: easysoft/zentao:latest
- **Description**: Open-source project management software for Agile/Scrum teams
- **Notes**: Official Docker image from EasySoft (ZenTao's parent company). ZenTao is a PHP-based ALM tool covering stories, tasks, bugs, cases, plans, and releases. Uses MySQL 8.0 as the database backend.
