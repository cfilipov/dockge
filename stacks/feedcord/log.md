# FeedCord

## Source
https://github.com/Qolors/FeedCord

## Notes
- Self-hosted RSS reader for Discord via webhooks
- Official Docker image: qolors/feedcord (tags: latest, beta, nightly)
- Requires appsettings.json config file with Discord webhook URL and RSS feeds
- No database needed; state persisted via PersistenceOnShutdown option
- Supports RSS and YouTube feed URLs
