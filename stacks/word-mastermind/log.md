# Word Mastermind

## Source
https://hub.docker.com/r/zuhkov/word-mastermind

## Description
Word Mastermind is a word-guessing game in the style of Mastermind. Players guess words and receive feedback on correct letters and positions. Similar to Wordle.

## Stack Details
- **app**: Word Mastermind web application (zuhkov/word-mastermind) on port 3000

## Notes
- Docker Hub image available from zuhkov/word-mastermind
- Simple single-container deployment, no database required
- All game state is client-side
