# Node-RED

## Source
- GitHub: https://github.com/node-red/node-red
- Docker Hub: nodered/node-red

## Description
Low-code programming tool for event-driven applications. Provides a browser-based flow editor for wiring together IoT devices, APIs, and online services.

## Stack
- **node-red**: Flow editor and runtime on port 1880
- Volume for persistent flows, credentials, and installed nodes
- Credential secret for encrypting sensitive flow data
