# SyncTube Stack

## Source
- GitHub: https://github.com/RblSb/SyncTube
- Category: Media Streaming - Video Streaming

## Description
SyncTube is a synchronized video watching platform that allows users to watch videos together in real time with chat support.

## Stack Components
- **synctube**: SyncTube application (Haxe/Node.js)

## Notes
- Single container setup
- Web UI on port 4200
- User configuration directory bind-mounted for persistence
- No external database required (file-based storage)
- Built with Haxe, compiled to JavaScript
