# Tiny File Manager

## Source
- GitHub: https://github.com/prasathmani/tinyfilemanager
- Docker Hub: https://hub.docker.com/r/tinyfilemanager/tinyfilemanager

## Description
The best web-based PHP file manager in a single file. Manage files efficiently with upload, download, edit, rename, copy, move, and delete capabilities.

## Stack
- tinyfilemanager/tinyfilemanager:latest — main application (port 80)

## Notes
- Default credentials: admin / admin@123 (or user / 12345)
- Single PHP file application
- Files managed in /var/www/html/data volume
- Supports file editing, code syntax highlighting, and file previews
