# EGroupware

## Overview
EGroupware is a web-based groupware server written in PHP. It provides email, calendar, contacts, tasks, project management, wiki, and more.

## Image
- **Docker Hub**: `egroupware/egroupware`
- **Source**: https://github.com/EGroupware/egroupware
- **Tags**: latest, 26, 26.2.20260224, 23.1, etc.

## Stack Details
- EGroupware PHP application with built-in web server
- MariaDB for database storage
- Persistent volumes for application data and file storage

## Notes
- Mature groupware platform (20+ years of development)
- Supports CalDAV/CardDAV for calendar and contact sync
- Built-in project management, wiki, knowledge base
- UTF-8 MB4 charset configured for full Unicode support
