# RomM

- **Source**: https://github.com/rommapp/romm
- **Category**: Games - Administrative Utilities & Control Panels
- **Description**: ROM manager and retro game library with metadata scraping and web UI
- **Docker image**: rommapp/romm:latest
- **Compose source**: Official examples/docker-compose.example.yml from master branch
- **Notes**: Web UI on :80 (mapped from internal :8080). MariaDB backend with built-in Redis. Supports ScreenScraper, RetroAchievements, SteamGridDB, and Hasheous metadata providers.
