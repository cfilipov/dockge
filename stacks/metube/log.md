# MeTube

- **Source**: https://github.com/alexta69/metube
- **Image**: `ghcr.io/alexta69/metube:latest`
- **Description**: Web GUI for yt-dlp with playlist support. Download videos from YouTube and dozens of other sites via a clean web interface.
- **Ports**: 8082 -> 8081
- **Volumes**: downloads
- **Key env vars**: MAX_CONCURRENT_DOWNLOADS, DEFAULT_THEME, DELETE_FILE_ON_TRASHCAN
- **Category**: Media Management
