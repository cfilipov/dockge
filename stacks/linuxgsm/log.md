# LinuxGSM

- **Source**: https://github.com/GameServerManagers/LinuxGSM
- **Category**: Games - Administrative Utilities & Control Panels
- **Description**: Command-line tool for deploying and managing dedicated game servers (120+ games)
- **Docker image**: ghcr.io/gameservermanagers/linuxgsm-docker:latest
- **Compose source**: Official LinuxGSM-Docker repository docker-compose.yml
- **Notes**: Set GAMESERVER env var to the desired server (e.g., csgoserver, valheimserver, mcserver). Ports vary by game.
