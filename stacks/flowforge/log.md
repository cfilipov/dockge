# FlowForge (FlowFuse)

## Source
- GitHub: https://github.com/FlowFuse/flowfuse
- Docker Hub: flowfuse/forge-k8s

## Description
Low-code platform built on Node-RED for managing and deploying Node-RED instances at scale. Provides team collaboration, DevOps pipelines, and centralized management.

## Stack
- **forge**: FlowForge application on port 3000
- **postgres**: PostgreSQL database for application data
- **broker**: Mosquitto MQTT broker for inter-instance communication
