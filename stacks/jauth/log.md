# Jauth

- **Category**: Web Servers / Auth Proxy
- **Source**: https://github.com/Jipok/Jauth
- **Image**: `jipok/jauth:latest`
- **Description**: Lightweight SSL/TLS reverse proxy with authorization via SSH keys or Telegram. Supports SSO, auto-generated self-signed certificates or Let's Encrypt, and whitelist-based access control.
- **Ports**: 80 (HTTP redirect), 443 (HTTPS), 2222 (SSH auth)
- **Config files**: `jauth.toml` (bind-mounted)
