# Immich Kiosk

- **Source**: https://github.com/damongolding/immich-kiosk
- **Docker image**: ghcr.io/damongolding/immich-kiosk:latest
- **Compose ref**: https://github.com/damongolding/immich-kiosk/blob/main/docker-compose.yaml
- **Description**: Slideshow/kiosk display for Immich photo libraries — designed for wall-mounted displays
- **Services**: immich-kiosk (Go app)
- **Notes**: Requires a running Immich instance. Config file at /config/config.yaml with API key and Immich URL. Supports person filtering, albums, date ranges.
