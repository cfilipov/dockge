# mStream

## Source
- GitHub: https://github.com/IrosTheBeggar/mStream
- Docker Image: lscr.io/linuxserver/mstream

## Description
mStream is a personal music streaming server. You can use mStream to stream your music from your home computer to any device, anywhere.

## Ports
- 3000: Web interface

## Volumes
- config: Application configuration
- music: Music library

## Notes
- LinuxServer.io image with PUID/PGID support
- Built-in web player with playlist support
- Supports FLAC, MP3, OGG, and other formats
