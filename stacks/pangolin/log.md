# Pangolin

- **Category**: Web Servers / Reverse Proxy
- **Source**: https://github.com/fosrl/pangolin
- **Image**: `fosrl/pangolin:latest`
- **Description**: Self-hosted tunneled reverse proxy with identity-aware access control. Designed for exposing internal services securely without port forwarding or VPN.
- **Ports**: 80 -> 3000 (HTTP), 443 -> 3001 (HTTPS)
- **Volumes**: `./config` for configuration
