# Taskwarrior (Taskchampion Sync Server)

- **Category**: Task Management & To-do Lists
- **GitHub**: https://github.com/GothenburgBitFactory/taskchampion-sync-server
- **Image**: agileit/taskchampion-sync-server
- **Description**: Sync server for Taskwarrior v3. Replaces the legacy taskd server. Clients sync task data via this server using the Taskchampion protocol.
- **Services**: taskchampion-sync-server (single container)
- **Ports**: 8080 (sync API)
- **Note**: This is a headless sync server; the Taskwarrior CLI is the client.
