# Retrom

- **Source**: https://github.com/JMBeresford/retrom
- **Category**: Games - Administrative Utilities & Control Panels
- **Description**: Centralized retro game library management with web client and in-browser emulation
- **Docker image**: ghcr.io/jmberesford/retrom-service:latest
- **Compose source**: Based on official docker/docker-compose.yml (adapted from build to image)
- **Notes**: API on :5101, web client on :3000. PostgreSQL backend. Supports EmulatorJS for in-browser play.
