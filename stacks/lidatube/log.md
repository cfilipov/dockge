# LidaTube

- **Source**: https://github.com/TheWicklowWolf/LidaTube
- **Image**: `thewicklowwolf/lidatube:latest`
- **Description**: Finds and fetches missing Lidarr albums via yt-dlp. Integrates with Lidarr API to identify missing music and download it from YouTube.
- **Ports**: 5001 -> 5000
- **Volumes**: config, downloads, localtime
- **Key env vars**: lidarr_address, lidarr_api_key, thread_limit, preferred_codec
- **Category**: Media Management
