# hitobito

- **Source**: https://github.com/hitobito/hitobito
- **Category**: Polls and Events
- **Description**: Community management solution for associations, clubs, and organizations with event management, contact management, and mailing lists.
- **Services**: hitobito (Rails app), worker (background jobs), db (MySQL), memcached
- **Notes**: No official Docker Compose upstream. Compose assembled from project architecture (Rails + MySQL + Memcached + Delayed::Job worker).
