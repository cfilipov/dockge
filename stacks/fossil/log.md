# Fossil

- **Source**: https://fossil-scm.org/
- **Image**: keyopt/fossil-scm (community multi-arch image)
- **Status**: created
- **Notes**: Fossil is a distributed version control system with built-in wiki, bug tracker, and web interface. No official Docker image exists. Using `keyopt/fossil-scm` community image. Serves repositories from `/repos` volume in repolist mode.
