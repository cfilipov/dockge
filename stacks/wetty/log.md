# WeTTY

- **Status**: created
- **Source**: https://github.com/butlerx/wetty
- **Image**: wettyoss/wetty:latest
- **Notes**: Web-based terminal emulator over SSH. Based on the official docker-compose.yml from the repo. Simplified to just the wetty service (removed the dev-only nginx and SSH containers). Connects to host SSH by default via host.docker.internal.
