# MediaGoblin

- **Source**: https://mediagoblin.org/ (GNU project)
- **Docker image**: mtlynch/mediagoblin:latest (community Docker image, ~451k pulls)
- **Compose ref**: Custom — no official compose; based on mtlynch/mediagoblin-docker
- **Description**: GNU MediaGoblin — federated media publishing platform for photos, video, and audio
- **Services**: mediagoblin (Python/Paste app with embedded SQLite)
- **Notes**: No official Docker image exists. Using mtlynch's community image (most popular). Default port 6543. Single-service deployment with SQLite.
