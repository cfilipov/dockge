# Lighttpd

- **Category**: Web Servers
- **Source**: https://hub.docker.com/r/sebp/lighttpd
- **Image**: `sebp/lighttpd:latest`
- **Description**: Lightweight, high-performance web server optimized for speed-critical environments. Low memory footprint and small CPU load make it suitable for servers with limited resources.
- **Ports**: 8080 -> 80 (HTTP)
- **Volumes**: `./htdocs` for web content
