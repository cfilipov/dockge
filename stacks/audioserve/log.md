# Audioserve

- **Source**: https://github.com/izderadicka/audioserve
- **Docker image**: izderadicka/audioserve
- **Reference**: README Docker Compose Example
- **Description**: Simple personal server to serve audio files from directories. Focused on audiobooks with tag caching support.
- **Ports**: 3000 (web UI)
- **Volumes**: config, audiobooks
