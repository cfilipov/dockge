# Tuleap

- **Status**: created
- **Source**: https://github.com/Enalean/tuleap
- **Image**: tuleap/tuleap-community-edition:latest
- **Description**: Full-featured open-source suite for application lifecycle management (ALM)
- **Notes**: Official Docker image from Tuleap. Includes MySQL 8.0 and Redis for session management. Tuleap provides Git, SVN, Kanban, Scrum, CI, and document management. The official compose from the repo is developer-oriented with build directives; this uses the production community-edition image instead.
