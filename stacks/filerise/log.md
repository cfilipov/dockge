# FileRise

## Source
- GitHub: https://github.com/error311/FileRise
- Docker Hub: https://hub.docker.com/r/error311/filerise-docker

## Description
Self-hosted web file manager and storage hub with WebDAV, sharing, per-folder ACLs, drag-and-drop uploads, OnlyOffice integration, and optional folder-level encryption at rest.

## Stack
- error311/filerise-docker:latest — main application (port 80)

## Notes
- PHP-based file manager with Vue.js frontend
- Supports chunked/resumable uploads, dual-pane mode, WebDAV
- Optional ClamAV integration for virus scanning
- Optional OnlyOffice integration for document editing
