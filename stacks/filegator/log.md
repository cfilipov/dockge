# FileGator

## Source
- GitHub: https://github.com/filegator/filegator
- Docker Hub: https://hub.docker.com/r/filegator/filegator

## Description
Powerful multi-user file manager with a single-page frontend built in Vue.js and a REST API backend in PHP.

## Stack
- filegator/filegator:latest — main application (port 8080)

## Notes
- Default credentials: admin / admin123
- Repository volume stores uploaded files
- Private volume stores configuration and session data
