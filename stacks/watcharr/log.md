# Watcharr

- **Source**: https://github.com/sbondCo/Watcharr
- **Image**: ghcr.io/sbondco/watcharr:latest
- **Port**: 3080
- **Description**: Self-hosted content watch list tracker. Track movies, TV shows, anime, and optionally video games with status, ratings, and progress.
- **Notes**: Built with Go + SvelteKit. Single volume for all data (database + cache). Supports dark/light themes. Optional IGDB integration for game tracking.
