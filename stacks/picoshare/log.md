# PicoShare

- **Source**: https://github.com/mtlynch/picoshare
- **Category**: File Transfer - Single-click & Drag-n-drop Upload
- **Description**: Minimalist, self-hosted web app for sharing files without restrictions on size, file type, or media format.
- **Image(s)**: `mtlynch/picoshare:latest`
- **Compose source**: Based on upstream README docker run command
- **Notes**: Single container with SQLite database stored in `/data` volume. Authentication via shared secret. Optional Litestream cloud replication support available.
