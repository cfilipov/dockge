# paaster

- **Source**: https://github.com/WardPearce/paaster
- **Image**: wardpearce/paaster:latest
- **Description**: End-to-end encrypted pastebin. Uses MongoDB for metadata and S3 (MinIO) for encrypted content.
- **Services**: App + MongoDB + MinIO
- **Ports**: 3015 (app), 9000 (MinIO API), 9001 (MinIO console)
- **Compose reference**: From upstream docker-compose.yml.
