# Apostrophe CMS

- **Source**: https://github.com/apostrophecms/apostrophe
- **Image**: node:18-alpine (no official pre-built Docker image)
- **Notes**: ApostropheCMS is a Node.js CMS that requires MongoDB. No official Docker image exists; the project recommends deploying to any Node.js platform. This compose uses a generic Node.js image with MongoDB for a plausible setup.
