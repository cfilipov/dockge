# Gladys Assistant

## Source
- GitHub: https://github.com/GladysAssistant/Gladys
- Docker Hub: gladysassistant/gladys

## Description
Privacy-first, open-source home assistant. Runs locally with support for Z-Wave, Zigbee, MQTT, and other protocols. Built with Node.js and SQLite.

## Stack
- **gladys**: Main application with host networking for device discovery
- Privileged mode for hardware access (Z-Wave/Zigbee dongles)
- Volume for SQLite database and configuration
