# Mikochi

## Source
- GitHub: https://github.com/zer0tonin/Mikochi
- Docker Hub: https://hub.docker.com/r/zer0tonin/mikochi

## Description
A minimalist remote file browser with upload support. Built with Go backend and React frontend.

## Stack
- zer0tonin/mikochi:latest — main application (port 8080)

## Notes
- Simple authentication via USERNAME/PASSWORD environment variables
- Files stored in /data volume
- Supports file upload, download, rename, delete
