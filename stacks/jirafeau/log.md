# Jirafeau

- **Source**: https://github.com/mojo42/Jirafeau
- **Category**: File Transfer - Single-click & Drag-n-drop Upload
- **Description**: Web application for uploading files and giving unique links to share them. Supports one-click upload, time-limited availability, and password protection.
- **Image(s)**: `mojo42/jirafeau:latest`
- **Compose source**: Created from Docker Hub image
- **Notes**: Single container PHP application. Data (uploads, links, async) stored in named volume. Setup wizard on first run.
