# A Dark Room

- **Source**: https://github.com/doublespeakgames/adarkroom
- **Description**: A Minimalist Text Adventure (web game)
- **Language**: JavaScript
- **Image**: nginx:alpine (serves the static HTML/JS game files)
- **Notes**: A Dark Room is a purely client-side web game. No official Docker image exists. This stack serves the game files via nginx. The `html/` directory would contain the game files cloned from the repository.
