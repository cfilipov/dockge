# Collabora Online Development Edition (CODE)

- **Status**: ok
- **Source**: https://github.com/CollaboraOnline/online
- **Image**: collabora/code:latest
- **Notes**: Collabora Online Development Edition provides LibreOffice-based online document editing. The `domain` env var accepts a regex of allowed WOPI host domains (escape dots with double backslash). SSL is disabled by default for use behind a reverse proxy with TLS termination. Admin console available at https://host:9980/browser/dist/admin/admin.html.
