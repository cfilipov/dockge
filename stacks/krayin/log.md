## Notes
- Uses the official published Docker image `webkul/krayin:2.1.0` from Docker Hub
- The devdocs show a compose with `build` context (from source); this compose uses the published image instead
- MySQL 8.0 database as specified in the official Docker documentation
- Default admin credentials: admin@example.com / admin123
- Access at http://localhost
