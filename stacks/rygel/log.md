# Rygel

- **Status**: skipped
- **Reason**: Rygel is a GNOME UPnP/DLNA media server that runs as a desktop application integrated with the GNOME desktop environment. No official Docker image or docker-compose configuration exists. It requires D-Bus, GStreamer, and GNOME libraries that make containerization impractical.
- **Source**: https://wiki.gnome.org/Projects/Rygel
