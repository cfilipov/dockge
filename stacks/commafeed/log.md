# CommaFeed

## Source
https://github.com/Athou/commafeed

## Notes
- Google Reader-inspired self-hosted RSS reader
- Official Docker image: athou/commafeed (Docker Hub)
- Available tags: latest, master-mysql-jvm, master-mysql, master-h2-jvm
- Supports PostgreSQL, MySQL, H2 databases
- Default port 8082
- Configured with PostgreSQL backend for production use
