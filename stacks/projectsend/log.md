# ProjectSend Stack

## Source
https://github.com/projectsend/projectsend

## Description
Self-hosted file sharing web application for clients. Allows uploading files and assigning them to specific clients/groups with notifications and access control.

## Services
- **projectsend** - Main application (lscr.io/linuxserver/projectsend:latest) on port 80
- **projectsend-db** - MariaDB 11 database

## Reference
Uses LinuxServer.io image for simplified deployment with PUID/PGID support.
