# SwingMusic

## Source
- GitHub: https://github.com/swingmx/swingmusic
- Docker Image: ghcr.io/swingmx/swingmusic

## Description
Swing Music is a beautiful, self-hosted music player for your local audio files. Like a cooler Spotify, but bring your own music.

## Ports
- 1970: Web interface

## Volumes
- music: Music library
- config: Application configuration

## Notes
- Modern, beautiful web-based music player
- Automatic library scanning and metadata fetching
- Supports albums, artists, playlists, and smart features
