# SIP Irrigation Control

## Source
- GitHub: https://github.com/Dan-in-CA/SIP

## Description
Sustainable Irrigation Platform — a Python-based irrigation controller for Raspberry Pi. Web interface for scheduling and controlling irrigation zones.

## Stack
- **sip**: Web application on port 8080
- Volume for persistent scheduling and zone data

## Notes
- No verified Docker Hub image; daninca/sip used as plausible image name
- Designed primarily for Raspberry Pi with GPIO access
