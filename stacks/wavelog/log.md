# Wavelog

- **Status**: created
- **Source**: https://github.com/wavelog/wavelog
- **Image**: philipreinken/wavelog:latest (Docker Hub, 1.7k+ pulls)
- **Notes**: Amateur radio logging software (successor to Cloudlog). The wavelog/wavelog repo does not provide an official Docker image; using the community philipreinken/wavelog image which is the most popular on Docker Hub. Uses MariaDB for the database backend.
