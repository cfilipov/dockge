# Ampache

- **Source**: https://github.com/ampache/ampache
- **Docker image**: ampache/ampache
- **Reference**: https://github.com/ampache/ampache-docker/blob/master/docker-compose.yml
- **Description**: Web-based audio/video streaming application and file manager. All-in-one image with built-in MySQL.
- **Ports**: 8080 (web UI)
- **Volumes**: config, logs, media, MySQL data
