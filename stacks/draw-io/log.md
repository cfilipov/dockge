# draw.io

- **Status**: ok
- **Source**: https://github.com/jgraph/drawio
- **Image**: jgraph/drawio:latest
- **Notes**: Self-hosted version of draw.io (diagrams.net), a web-based diagramming tool. Runs on Tomcat, serving on port 8080. No external database required. Diagrams are saved client-side (browser local storage, or exported to file).
