# AdventureLog

- **Source**: https://github.com/seanmorley15/AdventureLog
- **Description**: Self-hostable travel tracker and trip planner
- **Compose reference**: Official `docker-compose.yml` from repository root
- **Services**: frontend (SvelteKit), backend (Django), db (PostGIS)
- **Default ports**: 8015 (frontend), 8016 (backend API)
