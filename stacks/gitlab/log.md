# GitLab

- **Source**: https://gitlab.com/gitlab-org/gitlab
- **Image**: gitlab/gitlab-ce:latest (official Community Edition)
- **Status**: created
- **Notes**: Full-featured DevOps platform. Official Docker image from GitLab. Requires significant resources (4GB+ RAM recommended). Configured via GITLAB_OMNIBUS_CONFIG environment variable. shm_size set to 256m for Prometheus metrics.
