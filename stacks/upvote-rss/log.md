# Upvote RSS

- **Source**: https://github.com/johnwarne/upvote-rss
- **Image**: ghcr.io/johnwarne/upvote-rss:latest
- Generates rich RSS feeds from Reddit, Hacker News, Lemmy, Lobsters, and more.
- Based on Docker Compose example from project README.
- Reddit API credentials optional but recommended for higher rate limits.
