# cState

## Sources
- Official repo: https://github.com/cstate/cstate
- Official wiki Docker page: https://github.com/cstate/cstate/wiki/Docker
- Third-party Docker image repo: https://github.com/AethiQs-Felix/cstate_docker

## Notes
- cState is a Hugo theme for status pages, not a standalone application
- No official pre-built Docker image; the official Dockerfile requires building from source
- Using community image `ghcr.io/aethiqs-felix/cstate:latest` from AethiQs-Felix
- Compose based on the third-party repo's docker-compose example
- The official project recommends serverless deployment (Netlify) over Docker
