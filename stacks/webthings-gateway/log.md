# WebThings Gateway

## Source
- GitHub: https://github.com/WebThingsIO/gateway
- Docker Hub: webthingsio/gateway

## Description
Web of Things gateway implementation by Mozilla/WebThingsIO. Provides a web interface for monitoring and controlling smart home devices using the W3C Web of Things standard.

## Stack
- **gateway**: Main application with host networking
- HTTP on 8080, HTTPS on 4443
- Volume for persistent configuration and add-on data
