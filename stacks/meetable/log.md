# Meetable

- **Source**: https://github.com/aaronpk/Meetable
- **Category**: Polls and Events
- **Description**: Minimal event listing and calendar website. Supports Micropub, iCal feeds, and event RSVPs.
- **Services**: meetable (Laravel/PHP app), db (MySQL)
- **Notes**: No official Docker Compose upstream. Compose assembled from Dockerfile and Laravel architecture.
