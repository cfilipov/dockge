# Oddmuse Stack

## Source
- GitHub: https://github.com/kensanata/oddmuse
- Docker Hub: jamescherti/oddmuse

## Notes
- Oddmuse is a simple wiki engine written in Perl
- File-based storage (no database required)
- Community Docker image (jamescherti/oddmuse)
- Data stored in /opt/oddmuse/data
- Default port 8080
