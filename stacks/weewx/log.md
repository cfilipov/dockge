# WeeWX

- **Status**: created
- **Source**: https://github.com/weewx/weewx
- **Image**: felddy/weewx:latest (Docker Hub, 214k+ pulls)
- **Notes**: WeeWX is open-source weather station software. The weewx/weewx repo does not provide official Docker images. Using felddy/weewx which is the most popular community Docker image (also available as mitct02/weewx). The data volume stores weewx configuration and database.
