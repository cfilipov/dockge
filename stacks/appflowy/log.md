# AppFlowy Cloud

- **Category**: Task Management & To-do Lists
- **GitHub**: https://github.com/AppFlowy-IO/AppFlowy-Cloud
- **Image**: appflowyinc/appflowy_cloud
- **Description**: Self-hosted cloud backend for AppFlowy, a Notion alternative. Includes auth (GoTrue), object storage (MinIO), PostgreSQL, Redis, and admin frontend.
- **Services**: appflowy_cloud + gotrue + postgres + redis + minio + admin_frontend + nginx
- **Ports**: 80 (web via nginx)
