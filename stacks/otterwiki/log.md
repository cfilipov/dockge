# Otter Wiki Stack

## Source
- GitHub: https://github.com/redimp/otterwiki
- Docker Hub: redimp/otterwiki

## Notes
- An Otter Wiki is a simple wiki powered by Python and Git
- Git-backed storage with Markdown support
- Settings, database, and repository stored in ./app-data bind mount
- Default port 8080 (maps to container port 80)
