# Listaway

- **Category**: Task Management & To-do Lists
- **GitHub**: https://github.com/jeffrpowell/listaway
- **Image**: ghcr.io/jeffrpowell/listaway
- **Description**: Self-hosted shared list application for wishlists, tracking lists, and task lists. Supports authentication (email/password + OIDC), public sharing with randomized URLs, and group collaboration.
- **Services**: listaway + PostgreSQL
- **Ports**: 8080 (web UI)
- **Config**: init-db.sql creates the listaway schema on first run.
