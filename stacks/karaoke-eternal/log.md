# Karaoke Eternal

## Overview
Karaoke Eternal is an open karaoke party system with a browser-based interface.

## Image
- `radrootllc/karaoke-eternal:latest` (official, Docker Hub)

## Ports
- 8080 → 8080 (web UI)

## Volumes
- `ke_data` — configuration and database
- Media bind mount (read-only)
