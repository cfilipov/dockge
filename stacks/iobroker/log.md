# ioBroker

## Source
- GitHub: https://github.com/ioBroker/ioBroker
- Docker Hub: iobroker/iobroker

## Description
Integration platform for IoT that connects hundreds of adapters for smart home protocols, cloud services, and visualization. Admin UI on port 8081, web interface on 8082.

## Stack
- **iobroker**: Main application with admin and web UI
- Volume for persistent adapter data and configuration
