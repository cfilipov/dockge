# django-wiki Stack

## Source
- GitHub: https://github.com/django-wiki/django-wiki
- Docker Hub: camandel/django-wiki

## Notes
- django-wiki is a rewrite of django-simplewiki with a focus on extensibility
- Community Docker image (camandel/django-wiki)
- Uses PostgreSQL backend for persistence
- Admin user created via environment variables on first run
- Default port 8000
