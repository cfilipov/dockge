# XBackBone Stack

## Source
https://github.com/SergiX44/XBackBone

## Description
Lightweight PHP file manager supporting ShareX instant sharing. Handles images, GIF, video, code, formatted text uploads with multi-user management, search, and multiple storage backends (local, S3, GCS, Azure, Dropbox, FTP).

## Services
- **xbackbone** - Main application (lscr.io/linuxserver/xbackbone:latest) on ports 80/443

## Reference
Uses LinuxServer.io image. Web installer handles initial setup. Built-in SQLite database.
