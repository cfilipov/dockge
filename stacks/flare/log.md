# Flare

- **Source**: https://github.com/FlintSH/Flare
- **Category**: File Transfer - Single-click & Drag-n-drop Upload
- **Description**: Modern file uploader with NextAuth authentication and PostgreSQL backend.
- **Image(s)**: `flintsh/flare:latest`, `postgres:16-alpine`
- **Compose source**: Adapted from upstream `docker-compose.yml` (replaced `build:` with `image:`)
