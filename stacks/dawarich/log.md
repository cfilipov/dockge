# Dawarich

- **Source**: https://github.com/Freika/dawarich
- **Description**: Self-hosted location history and GPS track manager
- **Compose reference**: Official `docker/docker-compose.yml`
- **Services**: dawarich_app (Rails), dawarich_sidekiq (background jobs), dawarich_db (PostGIS), dawarich_redis (Redis)
- **Default port**: 3000
