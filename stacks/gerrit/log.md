# Gerrit

- **Source**: https://github.com/GerritCodeReview/gerrit
- **Image**: gerritcodereview/gerrit:latest (official)
- **Status**: created
- **Notes**: Web-based code review tool for Git repositories. Official Docker image from the Gerrit project. Exposes HTTP UI on 8080 and SSH on 29418. Uses H2 embedded database by default; production deployments typically use PostgreSQL/MySQL.
