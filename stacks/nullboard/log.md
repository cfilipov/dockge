# Nullboard

- **Category**: Task Management & To-do Lists
- **GitHub**: https://github.com/apankrat/nullboard (original), https://github.com/nicksitcompiled/nullboard (fork)
- **Image**: richardsoper/nullboard
- **Description**: Minimalist kanban board focused on compactness and readability. Originally a client-side-only HTML file; this Docker image wraps it with a lightweight server for persistence.
- **Services**: nullboard (single container)
- **Ports**: 8080 (web UI)
- **Note**: The original Nullboard is purely client-side. This uses a community Docker image that adds server-side persistence.
