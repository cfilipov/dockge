# myTinyTodo

- **Category**: Task Management & To-do Lists
- **GitHub**: https://github.com/maxpozdeev/mytinytodo
- **Image**: maxpozdeev/mytinytodo
- **Description**: Simple open-source to-do list script written in PHP. AJAX interface, multiple lists, drag-and-drop sorting, smart syntax for tags/priorities/dates. Uses SQLite by default.
- **Services**: mytinytodo (single container with Apache + PHP + SQLite)
- **Ports**: 8080 (web UI)
