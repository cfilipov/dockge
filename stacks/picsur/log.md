# Picsur Stack

## Source
https://github.com/CaramelFur/Picsur

## Description
Self-hosted image sharing and hosting platform with PostgreSQL backend. Supports image upload, conversion, and sharing with admin panel.

## Services
- **picsur** - Main application (ghcr.io/caramelfur/picsur-alpha:latest) on port 8080
- **picsur-postgres** - PostgreSQL 17 database

## Reference
Compose based on upstream `support/picsur.docker-compose.yml`.
