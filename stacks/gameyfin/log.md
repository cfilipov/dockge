# Gameyfin

- **Source**: https://github.com/gameyfin/gameyfin
- **Category**: Games - Administrative Utilities & Control Panels
- **Description**: Video game library manager (like Jellyfin for games), auto-scans and indexes game collections
- **Docker image**: ghcr.io/gameyfin/gameyfin:2
- **Compose source**: Official docker/docker-compose.example.yml from repository
- **Notes**: Web UI on :8080, mount game library folder. Supports plugins, OAuth2/OIDC, H2 database.
