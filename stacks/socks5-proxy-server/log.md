# socks5-proxy-server

- **Status**: ok
- **Image**: serjs/go-socks5-proxy
- **Note**: Simple SOCKS5 proxy server with optional authentication. Leave PROXY_USER and PROXY_PASSWORD empty to disable auth (not recommended for public-facing deployments). ALLOWED_IPS is a comma-separated list of IPs permitted to connect.
- **Source**: https://github.com/serjs/socks5-server
- **Docker Hub**: https://hub.docker.com/r/serjs/go-socks5-proxy
