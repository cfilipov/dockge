# Atomic Server

## Sources
- Official repo: https://github.com/atomicdata-dev/atomic-server
- Official docker-compose.yml: https://raw.githubusercontent.com/atomicdata-dev/atomic-server/develop/docker-compose.yml

## Notes
- Atomic Server is a linked data knowledge graph database with document/graph management
- Official image: `joepmeneer/atomic-server`
- Compose taken directly from the official docker-compose.yml in the repository
- Uses a named volume for persistent storage
- Web UI accessible on port 80
- The original compose references an .env file generated by `atomic-server generate-dotenv`, omitted here as it's optional
