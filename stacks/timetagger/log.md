# TimeTagger

## Source
- GitHub: https://github.com/almarklein/timetagger
- Image: ghcr.io/almarklein/timetagger

## Research
- Docker Compose from `deploy/docker-compose.yml` in repo
- Single service, no database needed (uses local file storage)
- Default credentials: test / test (generate your own at https://timetagger.app/cred)
- Web UI on port 80
- Also has a non-root variant at `deploy/docker-compose.nonroot.yml`
