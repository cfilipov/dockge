# Planka

- **Category**: Task Management & To-do Lists
- **GitHub**: https://github.com/plankanban/planka
- **Image**: ghcr.io/plankanban/planka
- **Description**: Elegant open-source project tracking tool with real-time kanban boards. Supports multiple projects, board members, labels, due dates, attachments, OIDC, and S3 storage.
- **Services**: planka + PostgreSQL
- **Ports**: 3000 (web UI)
