# Domoticz

## Source
- GitHub: https://github.com/domoticz/domoticz
- Docker Hub: domoticz/domoticz

## Description
Open-source home automation system supporting various protocols (Z-Wave, Zigbee, 433MHz, etc.) with a web-based interface for monitoring and controlling smart devices.

## Stack
- **domoticz**: Main application with web UI on port 8080, HTTPS on 8443
- Volume for persistent user data
- Device passthrough for USB controllers (Z-Wave/Zigbee sticks)
