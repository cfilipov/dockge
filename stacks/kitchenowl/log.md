# KitchenOwl

- **Source**: https://github.com/TomBursch/kitchenowl
- **Description**: Self-hosted grocery list and recipe manager with mobile apps
- **Architecture**: Flutter web frontend, Python/Flask backend with SQLite
- **Images**: tombursch/kitchenowl-web, tombursch/kitchenowl-backend
- **Compose reference**: Directly from upstream docker-compose.yml
- **Notes**: Simple two-container setup. Backend stores data in /data volume. Has companion mobile apps for iOS/Android.
