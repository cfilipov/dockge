# Maloja

- **Source**: https://github.com/krateng/maloja
- **Docker image**: krateng/maloja
- **Reference**: https://hub.docker.com/r/krateng/maloja
- **Description**: Self-hosted music scrobble database for personal listening statistics. Replacement for Last.fm with full data ownership.
- **Ports**: 42010 (web UI + API)
- **Volumes**: data (database and configuration)
- **Note**: MALOJA_SKIP_SETUP is required for non-interactive container operation.
