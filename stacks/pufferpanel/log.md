# PufferPanel

- **Source**: https://github.com/pufferpanel/pufferpanel
- **Category**: Games - Administrative Utilities & Control Panels
- **Description**: Open-source game server management panel with multi-user support and SFTP
- **Docker image**: pufferpanel/pufferpanel:latest
- **Compose source**: Created from official documentation
- **Notes**: Web UI on :8080, SFTP on :5657. Supports Docker-in-Docker for server isolation. Mount docker.sock for container management.
