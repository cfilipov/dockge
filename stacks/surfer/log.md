# Surfer

## Source
- Git: https://git.cloudron.io/cloudron/surfer
- Docker Hub: https://hub.docker.com/r/cloudron/io.cloudron.surfer

## Description
Simple static file server with a web UI to manage files. Part of the Cloudron ecosystem but usable standalone.

## Stack
- cloudron/io.cloudron.surfer:latest — main application (port 3000)

## Notes
- Node.js-based static file server
- Web UI for uploading and managing files
- Data stored in /app/data volume
