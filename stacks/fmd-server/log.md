# FMD Server (Find My Device)

## Source
- GitLab: https://gitlab.com/fmd-foss/fmd-server
- Website: https://fmd-foss.org

## Description
Server component for the FMD (Find My Device) Android app. Allows locating and controlling your Android devices remotely as a self-hosted alternative to commercial find-my-device services.

## Stack
- **fmd-server**: Go-based server on port 8080
- Volume for persistent data
