# PictShare Stack

## Source
https://github.com/HaschekSolutions/pictshare

## Description
Self-hosted image/gif/mp4/text hosting. 100% file-based with no database needed. Supports image filters, resizing, WebP conversion, and deduplication.

## Services
- **pictshare** - Main application (ghcr.io/hascheksolutions/pictshare:latest) on port 80

## Reference
Docker image from official GHCR. Simple single-container deployment.
