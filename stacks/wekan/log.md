# Wekan

- **Category**: Task Management & To-do Lists
- **GitHub**: https://github.com/wekan/wekan
- **Image**: ghcr.io/wekan/wekan
- **Description**: Open-source kanban board (Trello alternative). Supports boards, swimlanes, lists, cards, checklists, due dates, labels, attachments, REST API, webhooks, and many integrations.
- **Services**: wekan (Meteor/Node.js app) + MongoDB
- **Ports**: 80 (web UI)
