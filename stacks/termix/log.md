# Termix

## Sources
- Official repo: https://github.com/Termix-SSH/Termix
- Docker Compose example in README

## Notes
- Termix is a web-based SSH client available as web app, desktop app, PWA, and mobile app
- Official image: `ghcr.io/lukegus/termix:latest`
- Compose taken directly from the official README
- Data stored in named volume at /app/data (encrypted SQLite database)
- Web UI accessible on port 8080
