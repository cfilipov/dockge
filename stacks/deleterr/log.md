# Deleterr

- **Source**: https://github.com/rfsbraz/deleterr
- **Image**: `ghcr.io/rfsbraz/deleterr:latest`
- **Description**: Intelligent media library cleanup for Plex. Uses Radarr, Sonarr, and Tautulli to identify and delete media based on watch history, age, and streaming availability.
- **Ports**: None (runs on schedule)
- **Volumes**: config (contains settings.yaml)
- **Key env vars**: LOG_LEVEL
- **Category**: Media Management
