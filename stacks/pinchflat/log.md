# Pinchflat

- **Source**: https://github.com/kieraneglin/pinchflat
- **Image**: `ghcr.io/kieraneglin/pinchflat:latest`
- **Description**: Self-hosted YouTube media manager using yt-dlp. Downloads content from channels and playlists on a schedule. Supports Plex/Jellyfin/Kodi integration, RSS feeds, SponsorBlock.
- **Ports**: 8945 -> 8945
- **Volumes**: config, downloads
- **Key env vars**: TZ
- **Category**: Media Management
