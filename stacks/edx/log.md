# edX (Open edX)

- **Source**: https://github.com/openedx
- **Image**: treehouses/openedx:latest (community Docker image)
- **Description**: Open-source MOOC platform originally from MIT/Harvard. Django-based with MySQL, MongoDB, Redis.
- **Services**: lms (learner-facing), cms (Studio course authoring), mysql, mongodb, redis
- **Notes**: Official Open edX uses Tutor for deployment (very complex). This uses the treehouses community image as a simplified alternative. Full production setup would use Tutor.
