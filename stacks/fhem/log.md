# FHEM

## Source
- Docker Hub: fhem/fhem
- GitHub: https://github.com/fhem/fhem-docker

## Description
Perl-based home automation server supporting a wide range of devices and protocols. Web interface on port 8083, telnet on 7072.

## Stack
- **fhem**: Main server with web UI and telnet interface
- Configurable UID/GID for file permissions
- Volume for persistent FHEM data
