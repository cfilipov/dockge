# Geo2tz

- **Source**: https://github.com/noandrea/geo2tz
- **Description**: Privacy-friendly, self-hosted geo-coordinates to timezone API
- **Compose reference**: Constructed from README docker run examples
- **Services**: geo2tz (Go API server)
- **Default port**: 2004
- **API**: `GET /tz/{latitude}/{longitude}` returns timezone JSON
