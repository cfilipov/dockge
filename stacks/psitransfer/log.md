# PsiTransfer Stack

## Source
https://github.com/psi-4ward/psitransfer

## Description
Simple self-hosted file sharing solution. No accounts needed, supports resumable uploads/downloads via tus.io, one-time downloads, password protection, and zip/tar.gz archives.

## Services
- **psitransfer** - Main application (psitrax/psitransfer:latest) on port 3000

## Reference
Docker image from Docker Hub (psitrax/psitransfer). Data volume needs UID 1000.
