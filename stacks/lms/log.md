# LMS - Lightweight Music Server

- **Source**: https://github.com/epoupon/lms
- **Docker image**: epoupon/lms
- **Reference**: https://hub.docker.com/r/epoupon/lms
- **Description**: Self-hosted music streaming server with Subsonic/OpenSubsonic API support. Features tag-based browsing, recommendations, MusicBrainz/ListenBrainz integration, transcoding, and playlists.
- **Ports**: 5082 (web UI)
- **Volumes**: data (working directory), music (read-only)
