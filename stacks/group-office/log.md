# Group-Office

## Overview
Group-Office is a groupware and CRM platform by Intermesh. It provides email, calendar, contacts, tasks, file sharing, and project management.

## Image
- **Docker Hub**: `intermesh/groupoffice`
- **Source**: https://github.com/Intermesh/groupoffice
- **Tags**: latest, 26.0.13, 26.0, 25.0.91, 25.0

## Stack Details
- Group-Office PHP application with Apache
- MariaDB for database storage
- Persistent volumes for application data and configuration

## Notes
- Professional groupware with CRM capabilities
- Supports IMAP email, CalDAV, CardDAV
- Companion mailserver image available: `intermesh/groupoffice-mailserver`
- UTF-8 MB4 charset for full Unicode support
