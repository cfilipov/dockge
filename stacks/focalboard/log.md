# Focalboard

- **Category**: Task Management & To-do Lists
- **GitHub**: https://github.com/mattermost-community/focalboard
- **Image**: mattermost/focalboard
- **Description**: Open-source project management tool, alternative to Trello/Notion/Asana. Kanban boards, table views, calendar views. Originally by Mattermost, now community-maintained.
- **Services**: focalboard (single container with embedded SQLite)
- **Ports**: 8000 (web UI)
- **Note**: Project is archived/community-maintained; 2.6M+ Docker Hub pulls.
