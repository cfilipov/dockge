# Stretto

## Source
- GitHub: https://github.com/benkaiser/stretto

## Description
Stretto is a beautiful web-based music player that aggregates music from YouTube, Spotify, SoundCloud, and other sources into a single unified interface.

## Services
- **stretto**: Node.js web application
- **mongo**: MongoDB database

## Ports
- 3000: Web interface

## Volumes
- mongo-data: MongoDB data

## Notes
- Aggregates music from multiple streaming sources
- Web-based player with playlist support
- Requires MongoDB for data storage
