# HAProxy

- **Category**: Web Servers / Load Balancer
- **Source**: https://hub.docker.com/_/haproxy (Official Docker image)
- **Image**: `haproxy:lts`
- **Description**: Reliable, high-performance TCP/HTTP load balancer. The de facto standard open-source load balancer used by many high-traffic websites.
- **Ports**: 80 (HTTP), 443 (HTTPS), 8404 (Stats)
- **Config files**: `haproxy.cfg` (bind-mounted)
