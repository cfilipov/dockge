# Nginx Proxy Manager

- **Category**: Web Servers / Reverse Proxy
- **Source**: https://github.com/NginxProxyManager/nginx-proxy-manager
- **Image**: `jc21/nginx-proxy-manager:latest`
- **Description**: Docker-based reverse proxy with a beautiful web UI for managing NGINX proxy hosts, SSL certificates (Let's Encrypt), redirections, and streams. No NGINX config knowledge needed.
- **Ports**: 80 (HTTP), 443 (HTTPS), 81 (Admin UI)
- **Volumes**: Named volumes for data and Let's Encrypt certificates
- **Default login**: admin@example.com / changeme
