# Drupal Commerce

- **Source**: https://github.com/drupalcommerce/commerce
- **Status**: ok
- **Image**: drupal:11 + postgres:16-alpine
- **Notes**: Drupal Commerce is a Drupal module. This stack runs the official Drupal Docker image with PostgreSQL. Commerce module must be installed via Composer after setup (`composer require drupal/commerce`). Web UI on port 8080.
