# Open Source Routing Machine (OSRM)

- **Source**: https://github.com/Project-OSRM/osrm-backend
- **Description**: High-performance routing engine for shortest paths in road networks
- **Compose reference**: Constructed from README Docker examples
- **Services**: osrm (C++ routing backend)
- **Default port**: 5000
- **Notes**: Requires pre-processed OSM data (extract, partition, customize steps before running routed). Uses MLD algorithm by default.
