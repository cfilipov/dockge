# Plik Stack

## Source
https://github.com/root-gg/plik

## Description
Scalable temporary file upload system (like WeTransfer, self-hosted). Supports S3/GCS/Swift backends, OneShot downloads, stream mode, end-to-end encryption, and password protection.

## Services
- **plik** - Main application (rootgg/plik:latest) on port 8080

## Reference
Docker image from Docker Hub (rootgg/plik). Single-container with local file storage.
