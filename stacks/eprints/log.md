# EPrints Stack

## Status: SKIPPED

## Source
- Repository: https://github.com/eprints/eprints3.4
- Demo: http://tryme.demo.eprints-hosting.org/

## Research
- No Docker-related files in the GitHub repository (no Dockerfile, no docker-compose)
- No official Docker Hub image
- Only community images exist (e.g., kbai/eprints) but these are unofficial, outdated (Ubuntu 16.04), and lack compose files
- EPrints installation is a traditional server install via Makefile/configure

## Reason for Skip
No official Docker image or compose file exists. Cannot create a reliable compose stack from unofficial community images.
