# openHAB

## Source
- GitHub: https://github.com/openhab/openhab-core
- Docker Hub: openhab/openhab

## Description
Vendor and technology agnostic open-source home automation software. Supports hundreds of technologies and provides a unified interface for smart home devices.

## Stack
- **openhab**: Main application with host networking
- HTTP on 8080, HTTPS on 8443
- Three volumes: addons, configuration, and user data
- Unlimited crypto policy for full TLS support
