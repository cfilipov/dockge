# wanderer

- **Source**: https://github.com/open-wanderer/wanderer
- **Description**: Trail database for uploading recorded tracks or creating new ones with metadata
- **Compose reference**: Official `docker-compose.yml` from repository
- **Services**: web (SvelteKit frontend), db (PocketBase backend), search (Meilisearch)
- **Default ports**: 3000 (web UI), 8090 (PocketBase API), 7700 (Meilisearch)
