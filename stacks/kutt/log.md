# Kutt

## Sources
- GitHub repo: https://github.com/thedevs-network/kutt
- Docker Hub image: kutt/kutt
- docker-compose.yml found in repo root

## Notes
- Compose based on repo's default docker-compose.yml (SQLite variant)
- Also supports PostgreSQL, MariaDB with Redis (separate compose files in repo)
- Port 3000 exposed
- Custom themes can be mounted via the custom volume
