# Yamtrack

- **Status**: created
- **Source**: https://github.com/FuzzyGrim/Yamtrack
- **Image**: ghcr.io/fuzzygrim/yamtrack:latest
- **Notes**: Self-hosted media tracker for movies, TV shows, anime, manga, video games, books, comics, and board games. Compose based on the official docker-compose.yml from the FuzzyGrim/Yamtrack repo. Uses SQLite (bind-mounted ./db) and Redis for caching.
