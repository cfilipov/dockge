# WikiDocs Stack

## Source
- GitHub: Zavy86/WikiDocs
- Docker Hub: zavy86/wikidocs

## Description
WikiDocs is a databaseless Markdown flat-file wiki engine. Content is stored as Markdown files on disk — no database required. Simple, lightweight, and easy to deploy.

## Volumes
- `wikidocs-data` — Markdown wiki content (datasets/)

## Ports
- 8080 → 80 (HTTP)
