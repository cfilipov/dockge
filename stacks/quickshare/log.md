# QuickShare Stack

## Source
https://github.com/ihexxa/quickshare

## Description
Quick and simple file sharing between devices. Supports multi-user, resumable uploads/downloads, directory sharing, QR codes, fuzzy search, and per-user quotas.

## Services
- **quickshare** - Main application (hexxa/quickshare:latest) on port 8686

## Reference
Docker image from Docker Hub (hexxa/quickshare). Single container with file-based storage.
