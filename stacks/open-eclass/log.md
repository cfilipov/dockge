# Open eClass

- **Source**: https://github.com/gunet/openeclass
- **Image**: openeclass/php:latest (Docker Hub)
- **Description**: Greek open-source e-learning platform (formerly GUnet eClass). PHP/MySQL asynchronous eLearning platform.
- **Services**: eclass (PHP app with Apache), mariadb
- **Notes**: Developed by the Greek Universities Network (GUnet). Official Docker images on Docker Hub.
