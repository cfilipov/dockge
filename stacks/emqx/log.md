# EMQX

## Source
- GitHub: https://github.com/emqx/emqx
- Docker Hub: emqx/emqx

## Description
Highly scalable, distributed MQTT broker for IoT, IIoT, and connected vehicles. Supports MQTT 5.0/3.1.1, MQTT over WebSocket, and includes a management dashboard.

## Stack
- **emqx**: MQTT broker with dashboard on port 18083
- MQTT on 1883, MQTTS on 8883, WebSocket on 8083/8084
- Volumes for data persistence and logs
