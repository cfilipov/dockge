# Uguu Stack

## Source
https://github.com/nokonoko/uguu

## Description
Simple lightweight temporary file hosting and sharing platform. One-click upload, no registration, drag & drop support, ShareX compatible, with file extension and MIME filtering.

## Services
- **uguu** - Main application (ghcr.io/nokonoko/uguu:latest) on port 80

## Reference
PHP application with Nginx. Based on Pomf. Supports SQLite/MySQL/PostgreSQL backends.
