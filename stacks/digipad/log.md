# Digipad

## Overview
Digipad is a collaborative online pinboard tool by La Digitale. It allows creating multimedia walls for organizing ideas, resources, and notes.

## Image
- **Docker Hub**: `ladigitale/digipad` (not found on Docker Hub search; image name based on La Digitale naming conventions)
- **Source**: https://ladigitale.dev/digipad/
- **Codeberg**: https://codeberg.org/ladigitale/digipad

## Stack Details
- Single container Node.js application
- Persistent data volume for pad storage
- Configurable text length limits
