# Algernon

- **Category**: Web Servers
- **Source**: https://github.com/xyproto/algernon
- **Image**: `xyproto/algernon:latest`
- **Description**: Small self-contained pure-Go web server with Lua, Teal, Markdown, Pongo2, HyperApp, Amber, Sass, GCSS, JSX, BoltDB, Redis, PostgreSQL, MariaDB support. Serves HTTP/2 with auto-generated TLS certificates.
- **Ports**: 4000 (HTTP)
- **Volumes**: `./data` for served content
