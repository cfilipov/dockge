# sigal

- **Status**: skipped
- **Reason**: sigal is a static gallery generator CLI tool (like Hugo or Jekyll for photo galleries). It processes images and outputs static HTML — it is not a long-running server. While a Docker image exists (jeanfi/sigal), it is meant for one-shot gallery builds, not as a self-hosted service managed by Docker Compose.
- **Source**: https://github.com/saimn/sigal
