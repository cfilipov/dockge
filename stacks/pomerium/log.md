# Pomerium

- **Category**: Web Servers / Identity-Aware Proxy
- **Source**: https://github.com/pomerium/pomerium
- **Image**: `pomerium/pomerium:latest`
- **Description**: Identity and context-aware reverse proxy providing secure, clientless access to internal applications. Supports SSO with any identity provider (Google, Okta, Azure AD, etc.).
- **Ports**: 80 (HTTP), 443 (HTTPS)
- **Config files**: `config.yaml` (bind-mounted)
- **Notes**: Requires IDP (identity provider) configuration
