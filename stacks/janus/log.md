# Janus Gateway

- **Status**: created
- **Source**: https://github.com/meetecho/janus-gateway
- **Image**: canyan/janus-gateway (Docker Hub, community image)
- **Notes**: General-purpose WebRTC server/gateway. No official Docker image from Meetecho. Using canyan/janus-gateway community image. Exposes HTTP API (8088), HTTPS (8089), admin (7088), WebSocket (8188), and RTP ports.
