# RosarioSIS

- **Source**: https://github.com/francoisjacquet/rosern (note: user asked for "rosern" but actual project is rosariosis)
- **Image**: rosariosis/rosariosis:latest (Docker Hub, official)
- **Description**: Free student information system (SIS). PHP/PostgreSQL app for school management: scheduling, grades, attendance, discipline, food service.
- **Services**: rosariosis (PHP app with Apache), postgres
- **Notes**: Official Docker image from rosariosis organization. Uses PostgreSQL (not MySQL).
