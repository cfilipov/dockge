# TRIP

- **Source**: https://github.com/itskovacs/trip
- **Description**: Minimalist POI map tracker and trip planner, self-hosted
- **Compose reference**: Official `docker-compose.yml` from repository
- **Services**: app (FastAPI Python backend)
- **Default port**: 8080 (mapped to internal 8000)
- **Notes**: SQLite-based, stores data in /app/storage volume
