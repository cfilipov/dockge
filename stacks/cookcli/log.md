# CookCLI

- **Source**: https://github.com/cooklang/CookCLI
- **Status**: SKIPPED - CLI-only tool
- **Reason**: CookCLI is a command-line tool for working with Cooklang recipes. It has an optional `cook server` subcommand for a local web UI, but it's distributed as a standalone binary (Homebrew, GitHub releases) with no Docker image or Docker Compose setup. Not suitable as a self-hosted Docker stack.
