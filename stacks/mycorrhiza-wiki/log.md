# Mycorrhiza Wiki Stack

## Source
- GitHub: https://github.com/bouncepaw/mycorrhiza
- Docker Hub: kytta/mycorrhiza

## Notes
- Mycorrhiza Wiki is a lightweight wiki engine with Mycomarkup
- Written in Go, filesystem-based (no database)
- Community Docker image (kytta/mycorrhiza)
- Wiki data stored in /wiki inside the container
- Default port 1737
