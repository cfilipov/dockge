# Moodle

- **Source**: https://moodle.org / https://github.com/moodle/moodle
- **Image**: bitnami/moodle:4 (Docker Hub, official Bitnami image)
- **Description**: The world's most popular open-source LMS. PHP/MySQL stack with extensive plugin ecosystem.
- **Services**: moodle (Bitnami PHP/Apache), mariadb (Bitnami MariaDB)
- **Notes**: Uses Bitnami images for easy setup. First boot runs installer automatically. Ports 8080/8443 (Bitnami convention).
