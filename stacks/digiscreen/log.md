# Digiscreen

- **Source**: https://codeberg.org/ladigitale/digiscreen (La Digitale)
- **Image**: node:20-alpine (generic Node runtime)
- **Description**: Interactive classroom screen tool by La Digitale. Displays timers, text, drawings, etc. for classroom projection.
- **Services**: digiscreen (Node.js app)
- **Notes**: No official Docker image. Uses generic Node image following La Digitale convention (same pattern as digibuzzer, digipad, etc.).
