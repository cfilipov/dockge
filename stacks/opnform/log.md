# OpnForm

- **Source**: https://github.com/JhumanJ/OpnForm
- **Category**: Polls and Events
- **Description**: Open-source form builder with AI-powered form generation, conditional logic, and integrations.
- **Services**: api (Laravel/PHP), api-worker (queue), ui (Nuxt.js client), redis, db (PostgreSQL), ingress (nginx)
- **Notes**: Compose derived from upstream docker-compose.yml. Nginx ingress routes /api to backend, / to frontend.
