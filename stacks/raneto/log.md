# Raneto Stack

## Source
- GitHub: ryanlelek/Raneto
- Docker Hub: raneto/raneto

## Description
Raneto is an open-source Markdown-powered knowledgebase platform. Content is stored as static Markdown files on disk — no database required.

## Volumes
- `./content` — Markdown knowledge base articles
- `./config` — Raneto configuration files

## Ports
- 3000 → 3000 (HTTP)
