# Tamari

- **Source**: https://github.com/alexbates/Tamari
- **Description**: Full-featured recipe manager with explore database of 107k+ recipes, shopping lists, and meal planning
- **Architecture**: Python/Flask with SQLite, single container
- **Images**: alexbates/tamari (also on ghcr.io/alexbates/tamari)
- **Compose reference**: Based on Docker run command from README (no official docker-compose provided)
- **Notes**: Single container app. Features include recipe sharing, meal planner, shopping lists with barcode scanning, REST API, 107k+ explorable recipes, and customizable themes. Mail settings are optional for password reset.
