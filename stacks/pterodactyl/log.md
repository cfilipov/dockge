# Pterodactyl

- **Source**: https://github.com/pterodactyl/panel
- **Category**: Games - Administrative Utilities & Control Panels
- **Description**: Open-source game server management panel with user isolation and resource management
- **Docker image**: ghcr.io/pterodactyl/panel:latest
- **Compose source**: Official docker-compose.example.yml from develop branch
- **Notes**: Full stack with MariaDB + Redis + Panel. Requires Wings daemon on game server nodes. Web UI on :80/:443.
