# StatPing.ng

## Sources
- Official repo: https://github.com/statping-ng/statping-ng
- Official docker-compose.yml in repo root
- Docker Hub: https://hub.docker.com/r/adamboutcher/statping-ng

## Notes
- StatPing.ng is an open-source status page and monitoring tool
- Official image: `adamboutcher/statping-ng:latest`
- Supports SQLite (default), PostgreSQL, and MySQL backends
- Compose taken directly from the official docker-compose.yml (converted to Compose V2, removed version field)
- Additional compose files available for SSL and nginx setups in the dev/ directory
