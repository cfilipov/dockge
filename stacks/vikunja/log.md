# Vikunja

- **Category**: Task Management & To-do Lists
- **GitHub**: https://github.com/go-vikunja/vikunja
- **Image**: vikunja/vikunja
- **Description**: Open-source to-do app written in Go. Full-featured with projects, kanban boards, Gantt charts, CalDAV, reminders, assignees, labels, priorities, file attachments, and API. Self-contained frontend + API in single image.
- **Services**: vikunja + MariaDB
- **Ports**: 3456 (web UI + API)
