# Spooty

- **Source**: https://github.com/Raiper34/spooty
- **Image**: raiper34/spooty:latest
- **Port**: 3000
- **Description**: Self-hosted Spotify downloader. Downloads tracks/playlists/albums via Spotify metadata + YouTube. Can subscribe to playlists and auto-download new releases.
- **Notes**: Requires Spotify Developer app credentials (Client ID + Secret). Redirect URI must be configured to http://127.0.0.1:3000/api/callback. Built with NestJS + Angular.
