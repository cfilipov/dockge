# Polaris

## Source
- GitHub: https://github.com/agersant/polaris
- Docker Image: ogarcia/polaris

## Description
Polaris is a music streaming application, designed to let you enjoy your music collection from any computer or mobile device.

## Ports
- 5050: Web interface

## Volumes
- music: Music library (read-only)
- data: Application data and database

## Notes
- Web-based UI for browsing and playing music
- Supports multiple music directories
- User management with access control
- Mobile-friendly responsive interface
