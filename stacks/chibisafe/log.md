# Chibisafe

- **Source**: https://github.com/chibisafe/chibisafe
- **Category**: File Transfer - Single-click & Drag-n-drop Upload
- **Description**: Blazing fast file uploader and awesome bunker written in node. Supports multiple file upload, albums, and user accounts.
- **Image(s)**: `chibisafe/chibisafe:latest`
- **Compose source**: Based on Docker Hub image and project documentation
- **Notes**: Single container with SQLite database. Uploads and database stored in named volumes.
