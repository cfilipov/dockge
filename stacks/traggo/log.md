# Traggo

## Source
- GitHub: https://github.com/traggo/server
- Docker Hub: traggo/server
- Docs: https://traggo.net/install/

## Research
- Docker Compose from official install documentation at traggo.net
- Single service with SQLite storage (no external database needed)
- Image: `traggo/server:latest`
- Default credentials: admin / admin
- Web UI on port 3030
- Multi-arch: amd64, arm64, armv7
