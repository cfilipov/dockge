# Hoppscotch Community Edition

- **Status**: ok
- **Source**: https://github.com/hoppscotch/hoppscotch
- **Image**: hoppscotch/hoppscotch:latest
- **Notes**: Open-source API development ecosystem (alternative to Postman). Uses the all-in-one (AIO) image which bundles frontend app, admin dashboard, and backend in a single container. Requires PostgreSQL. DATA_ENCRYPTION_KEY must be exactly 32 characters. Update VITE_*_URL variables when deploying behind a domain.
