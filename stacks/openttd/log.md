# OpenTTD

- **Source**: https://github.com/OpenTTD/OpenTTD
- **Description**: Open-source transport business simulation game server
- **Image**: bateau/openttd:latest (tags: latest, 15.2, 15.1)
- **Ports**: 3979/tcp + 3979/udp (game server)
- **Notes**: OpenTTD is an open-source simulation game based on Transport Tycoon Deluxe. The bateau/openttd image provides a dedicated server. An alternative image is redditopenttd/openttd. Configuration is via openttd.cfg which controls server settings, map generation, and difficulty. Save games are persisted in a named volume.
