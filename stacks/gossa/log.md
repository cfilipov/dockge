# Gossa

## Source
- GitHub: https://github.com/pldubouilh/gossa
- Docker Hub: https://hub.docker.com/r/pldubouilh/gossa

## Description
A fast and simple multimedia file server. Lightweight Go-based file browser with upload support and media playback.

## Stack
- pldubouilh/gossa:latest — main application (port 8001)

## Notes
- Serves files from /shared directory
- Supports upload, download, rename, delete, and media playback
- Minimal resource usage — single Go binary
