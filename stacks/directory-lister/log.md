# Directory Lister

## Source
- GitHub: https://github.com/DirectoryLister/DirectoryLister
- Docker Hub: https://hub.docker.com/r/directorylister/directorylister

## Description
The easiest way to expose the contents of any web-accessible folder for browsing and sharing. PHP-based directory listing application.

## Stack
- directorylister/directorylister:latest — main application (port 80)

## Notes
- Mount your files to /var/www/html/app for listing
- Configurable via .directory-lister.php config file
