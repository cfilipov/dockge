# Bitwarden

## Sources
- GitHub server repo: https://github.com/bitwarden/server
- Self-host repo: https://github.com/bitwarden/self-host
- Docker Compose file: https://github.com/bitwarden/self-host/blob/main/bitwarden-lite/docker-compose.yml
- Settings template: https://github.com/bitwarden/self-host/blob/main/bitwarden-lite/settings.env

## Notes
- Using the "bitwarden-lite" unified image from the self-host repo (simpler than the multi-container setup)
- The full Bitwarden self-hosted deployment uses 10+ containers generated by install scripts
- The lite image bundles all services into a single container with supervisord
- Supports MariaDB, PostgreSQL, MS SQL, or SQLite backends; using MariaDB per the default example
- Replaced variable substitution `${REGISTRY:-ghcr.io/bitwarden}` and `${TAG:-...}` with direct values
- Requires settings.env with domain, DB config, and installation credentials from https://bitwarden.com/host/
